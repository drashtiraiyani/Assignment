18. What is difference between HTML and XHTML?

    HTML:

  • Hyper Text Markup Language
  • It is saved with extension .htm/.html
  • It is not case-sensitive
  • It allows overlapping tags
  • It is extended from SGML
  • Atrributes values are not significant in HTML
  • All content must be included in Body tag.

    • XHTML:

  • Extensible Hyper Text Markup Language
  • It is saved with extension .xht/.xhtml/.xml
  • It can be written only in lower case
  • It doesnot allow overlapping tags
  • It is extended from HMTL and XML
  • Attributes values are very important in XHTML
  • All content must be included in Blocks
